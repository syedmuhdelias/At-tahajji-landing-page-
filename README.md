# aT-Tahajji Promotional Website

This is a professional landing page for promoting a Quran reading class.

## Files
- `index.html` - Main website page
- `styles.css` - Full responsive styling
- `script.js` - Mobile menu and FAQ interaction

## Run locally

**Option A — quick preview (no install):** double-click `index.html`, or open it in Chrome/Edge/Firefox.

**Option B — local server (recommended):**

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:3000`).

## How to use
1. Open the site in your browser (see **Run locally** above).
2. Replace the placeholder WhatsApp number `60123456789` with your real number.
3. Replace `hello@attahajji.com` with your real email.
4. Update class information, prices, schedule, location, and teacher details if needed.

## Deploy to Vercel
You can deploy this as a static website by uploading the folder to GitHub and importing it into Vercel.
